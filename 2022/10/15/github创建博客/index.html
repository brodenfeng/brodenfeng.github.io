github创建博客

工具准备

在本地电脑上执行

安装nodjs

  1. 下载并安装工具包

下载地址 https://nodejs.org/en/download/

下载工具后双击安装包一步步安装即可。

https://nodejs.org/dist/v16.18.0/node-v16.18.0-x64.msi

  1. 检查安装结果

    C:\Users\fzb>node -v
v16.18.0

安装git客户端

参考文档 https://hexo.io/zh-cn/docs/

  1. 下载并安装工具包

下载地址 https://git-scm.com/download/

​ 下载工具后双击安装包一步步安装即可。

https://github.com/git-for-windows/git/releases/download/v2.38.0.windows.1/Git-2.38.0-64-bit.exe

  1. 检查安装结果

    C:\Users\fzb>git --version
git version 2.7.2.windows.1

Hexo下载

  1. 下载并安装Hexo

    C:\Users\fzb>npm install hexo-cli -g

github准备

创建github账号及仓库

  1. 创建一个github账号

    https://github.com/

  2. github上创建一个公开仓库，仓库名称和你的用户名一致,仓库名称为

    用户名.github.io

配置本地免密访问github

  1. 本地进入一个文件夹，如D:\blog,鼠标右键 弹出内容选中git bash here,进入git客户端命令窗口。

  2. 本地生成ssh秘钥对

    注意：不要用rsa算法，用ed25519算法，rsa算法被github认为不安全，会导致你无法做免密推送文件

$ ssh-keygen -t ed25519 -C "zhibfeng@126.com"

  3. 按提示到/c/Users/fzb/.ssh/id_rsa.pub路径打开该秘钥公钥复制到github

写一篇自己的博客并发布

将写好的博客文档，如Markdown 和 HTML 文件会被解析并放到D:\blog\source_posts目录即可

本地博客发布到github

安装发布工具

fzb@LAPTOP-U0PQ03U0 MINGW32 /d/blog
$ npm install hexo-deployer-git --save

added 1 package, and audited 243 packages in 10s

22 packages are looking for funding
  run `npm fund` for details

found 0 vulnerabilities

配置Hexo发布地址

修改 _config.yml文件deploy参数

deploy:
  type: git
  repo: git@github.com:brodenfeng/brodenfeng.github.io.git
  branch: main

注意：repo建议用ssh的地址，即git@github.com:brodenfeng/brodenfeng.github.io.git，如果用https的地址会在推送弹出输入账号密码窗口，你要手动输入账号和访问Token(令牌)。
